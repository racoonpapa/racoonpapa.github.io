Windows 11이 나온다고?

2015년 마이크로소프트는 '윈도우 10은 마이크로소프트의 PC용 마지막 운영체제'라는 말을 했다. (이례적으로 마이크로소프트에서 공식적으로 확인까지 해주었다.) Windows 7이나 Windows 8에서 업그레이드를 주저하는 사람들에게 '무료로 업그레이드'라는 초강수까지 두면서 마지막 운영체제라는 이야기까지 곁들이니 이에 대한 파급효과는 꽤 강력했다고 생각한다. 이전까지 마이크로소프트는 윈도우에 3.1, 95, 98, 2000, XP, Vista, 7, 8과 같은 알 수 없는 이름을 사용하며 2~3년마다 한 번씩 나타나 세금 걷듯이 업그레이드 비용을 받아왔는데 '윈도우 10이 마지막 운영체제'라는 말은 6년 동안 잘 지켜졌고.......
