커맨드 라인으로 웹사이트 열기 및 키오스크 모드 실행하기

(터미널이 뭔지 모른다, 이걸 왜 쓰냐 하면 그냥 '뒤로 가기'를 누르자.) 윈도우 Windows에서는 start 명령어를 사용해서 웹브라우저를 실행할 수 있는데 start 명령어의 /max, /min과 같은 파라미터는 동작하지 않지만 그럭저럭 유용하게 사용할 수 있다. 크롬을 사용하는 경우에는, 위와 같이 naver.com의 자리에 본인이 브라우징을 원하는 주소를 적으면 된다.
