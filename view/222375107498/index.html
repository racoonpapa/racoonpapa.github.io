Sublime Text 4 릴리즈

2015년쯤에 HTML / CSS / Javascript 편집기로 꽤나 좋다고 해서 구입한 이래로 지금까지 줄곧 주요 텍스트 에디터로 사용하던 서브라임 텍스트(Sublime Text)가 며칠 전에 뜬금없이 업데이트가 있다고 업그레이드하겠냐고 묻길래, '상당히 오랫동안 업데이트하지 않던 3버전이 유지 보수 업데이트를 해주나 보다'라고 생각하고 업데이트 버튼을 눌렀더니 3이 아니라 4로 변경되었다. 4.0은 GPU 렌더링을 지원하고 M1에 대응되며 파이썬 버전도 3.x에 대응시킨... 일일이 열거하기도 어려울 정도로 많은 기능이 추가되었다. 그러나 4.0의 가격은 80달러로 책정되었는데, 개인적으로 2021년에 텍스트 에디터를 80달러(약 9만 원) 주고 구.......
