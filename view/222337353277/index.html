파워토이즈(Microsoft PowerToys) 소개

아주 오래전에 Windows 95시절에도 마이크로소프트는 파워토이즈라는 유틸리티를 제공한 적이 있었는데 윈도우에서는 제공되지 않는 몇 가지 유용한 유틸리티(나는 Command Prompt Here를 유용하게 사용했었다)를 제공함으로써 조금 더 윈도우를 편리하게 쓸 수 있게 만들어주었다. 오랫동안 파워토이즈는 '그런 유틸리티가 있었지' 정도로 잊고 있었는데 작년에 새로운 Windows Terminal을 보다가 윈도우 10용의 파워토이즈가 오픈소스 프로젝트로 제작되고 있다는 것을 알게 되었고 가려운 부분을 살짝 긁어주는 유틸리티로 애용 중이다. 그래서 오늘은 마이크로소프트의 윈도우 10용 파워토이즈에 대해 소개해볼까 한다. 설치는 위.......
