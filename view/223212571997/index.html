다양한 언어 훑어보기 - 준비

보통 백엔드에서 사용하는 각종 언어에 대해 가볍게 훑어보는 포스팅. 심도 있는 주제는 아니고, 자신이 사용하지 않는 언어도 대충 이런 느낌으로 프로그래밍 되는구나를 보면 되겠다. 시작은 아마도 Java가 될 테고, C#, Python, Javascript, Go, Rust 정도를 다루게 될 예정이다. (C++은 코드가 너무 장황해서 뺐다.) REST API 류를 다루다 보면 가장 자주 다루는 부분은 JSON을 해석해서 자신의 데이터 포맷으로 담고, 가공된 이후에 JSON으로 바꾸어 반환하는 부분일 것이다. 딱 이 부분만 구현하는 것을 최대한 간단하게 살펴보겠다. (강좌 아니다) 본격적 코딩 이전에 데이터부터 생성했다. 이건 자바스크립트가 가장 편리하게 만들 수 있.......
