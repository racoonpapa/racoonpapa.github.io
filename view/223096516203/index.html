CNCF Landscape

Container Registry를 고민하다가 우연히 발견한 사이트를 와드 겸 해서 포스팅. Cloud Native Computing Foundation이라는 사이트에서 클라우드 네이티브 인터랙티브 랜드스케이프라는 이름으로 올려둔 것인데 원래 이쪽 분야가 아니다 보니 이제서야 알게 되었지만 여하튼 유용해 보인다. 가이드를 보면 목표는 모든 클라우드 네이티브 오픈 소스 프로젝트와 독점 제품을 분류별로 나누어 에코 시스템을 한눈에 볼 수 있게 만드는 것이라고 나와 있는데 자세한 내용은 아래의 링크를 참조하면 된다. 내 경우에 컨테이너 레지스트리로 쓸만한 것을 찾고 있었는데 Harbor, Dragonfly가 많이 사용되는 오픈소스 솔루션이라는 것을 쉽게 찾을 수 있.......
