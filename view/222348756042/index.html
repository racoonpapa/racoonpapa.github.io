M1 맥북 초기화 문제

이 문제가 항상 발생하는 건지 아닌지는 모르겠으나 맥을 사용하면서 초기화를 한두 번 해본 것도 아닌데, 굉장히 당혹스러운 경험을 해서 써본다. 문제 해결을 위해서 검색을 하다 보니 이 문제와 관련된 상당히 많은 수의 포스팅을 볼 수 있었는데, M1의 문제라기보다는 강화된 보안과 관련된 이슈로 생각된다. 우선 M1 맥북을 초기화하기 위해서는 기존의 Cmd + R을 비롯한 핫키를 누르면서 부팅을 하는 방식이 아니라 파워 버튼(Touch ID 버튼, 즉 지문 인식 버튼)을 부팅 시에 꾹 누르고 있으면 된다. 부팅 신호음이 들린 이후 일정 시간을 누르고 있으면 '좀 더 누르고 있으면 부팅 옵션이 활성화된다.'라는 식의 메시지가 화면.......
