다양한 언어 훑어보기 - 고(Go)

이 포스팅 시리즈는 앞서 말했지만 다양한 언어(백엔드에서 주로 사용되는 언어 위주)들을 가볍게 훑어보는 것이 목적이다. 프로그래밍 언어들이 어떤 특징들이 있는지 아주 얕은 정도로만 살펴보는데 프로그램이 공통적으로 수행하는 작업과 사용하는 데이터를 생성하는 스크립트는 다음의 링크에서 참고하기 바란다. 수업이나 튜토리얼이 아니기 때문에 언어에 대한 자세한 설명은 없고 각 언어별로 짧은 내 생각에 대해서는 코멘트를 넣도록 하겠다. Go Go 언어는 2009년에 만들어진 언어 쪽에서는 꽤 신제품에 해당하는 언어이다. 2009년에 나온 게 어째서 신제품?이라는 생각을 하는 사람도 있겠으나 프로그래밍 언어 자체가 새로운 것을 배.......
