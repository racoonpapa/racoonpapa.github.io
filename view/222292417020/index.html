스위프트 3 루시엔 모델 스펙을 보고...

지난 금요일인가? 스위프트 3 루시엔 모델 예약판매가 등록된 것을 알고 해당 사이트를 방문해보았는데 제품이 상당히 괜찮아 보인다. CPU가 5500U인데 뭔가 기시감이 들어서 내가 작년에 쓴 글을 보니 판매 패턴이 뭔가 반복되는 건가는 생각도 들었다. 작년에 내가 윈도우 노트북-사실은 리눅스용 노트북을 골랐다-을 고를 때에도 스위프트 3 르누아르 4500U 모델이 있어서 구입을 고려했던 적이 있었는데 당시에 정말 다 괜찮다고 생각했는데 250니트 디스플레이, 45% NTSC 보고 마음을 접었다. NTSC 45%니 72%니 하면 '그게 뭔데? 씹덕아!'라고 할 것 같으니 좀 더 와닿게 설명하자면, NTSC 45%는 '뭔가 뿌옇고 물빠진 것 같.......
